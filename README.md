### Installation

this works for both frontend-tsc and backend just changes the directory and install packages and run it locally.

1. Clone the repo
   ```sh
   git clone https://github.com/michaelnadar/tic_tac-tsc.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3.Run It Locally
   ```sh
   npm run dev
   ```
